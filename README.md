# Sample App following
## [*Michael Hartl's Ruby on Rails Tutorial - Learn Web Development with Rails 4th Edition*](http://railstutorial.org)

To set it up please

install the gems

´´´
$ bundle install --without production
´´´

update the database

´´´
$ rails db:migrate
´´´

run the test suite

´´´
$ rails test
´´´

and run the server

´´´
$ rails server
´´´
